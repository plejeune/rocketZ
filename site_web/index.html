RocketZ est une petite simulation de pilotage, jouable sur navigateur web et conçue en C# à l'aide du moteur Unity et API REST (Firebase), pour l'Epreuve intégrée du B.E.S. Webdeveloper 2019-2020.


Technologies utilisées

C#, Javascript, JSON
API REST, Firebase (Google)
HTML, CSS, JQuery, Bootstrap
Visual Studio, Brackets
